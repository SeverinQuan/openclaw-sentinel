# Usage

## Quick Start

```bash
bash scripts/install.sh
bash scripts/validate.sh
```

## Expected Behavior
- Default web queries use DuckDuckGo through `main`.
- Complex research routes to `web-researcher`.

## Manual Checks
- Review `presets/main.agent.md` and `presets/web-researcher.agent.md`.
- Review `presets/routing-policy.md` for delegation rules.
